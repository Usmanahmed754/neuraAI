# neuraAI
NeuraAI is a multi-LLM intelligent AI assistant built using Python and Streamlit. It integrates Gemini and Groq models with smart routing, tool usage (web search, calculator), and chat memory to deliver a ChatGPT-like experience.
NeuraAI 🧠

NeuraAI is an advanced AI assistant system that leverages multiple Large Language Models (Gemini + Groq) to provide intelligent, context-aware responses.

Unlike a simple chatbot, NeuraAI is designed as a mini AI agent system with tools, memory, and smart routing capabilities.

🔹 Features:
- Multi-LLM support (Gemini + Groq fallback system)
- Smart query routing (AI, Web Search, Calculator)
- Conversational memory with topic-based chats
- Streamlit-based interactive UI
- Real-world AI agent architecture

🔹 Tech Stack:
Python | Streamlit | Gemini API | Groq API | LLMs | Prompt Engineering

🔹 Key Learning:
- Building AI agent systems from scratch
- Working with multiple LLM APIs
- Prompt engineering techniques
- Designing conversational AI applications

This project demonstrates how modern AI systems combine multiple models and tools to create intelligent assistants.
